## Prerequisites

While each workshop may have its own specific prerequisites, the following are common requirements:

### General Requirements

1. You must be connected to the internet
2. You must be signed up for a [Microsoft Fabric account](https://aka.ms/tryfabric)
3. You must have a premium capacity backed workspace, through one of the following methods:
    - **FT** - [Fabric trial](https://learn.microsoft.com/fabric/get-started/fabric-trial)
    - **F** - [Fabric capacity](https://learn.microsoft.com/fabric/enterprise/buy-subscription#buy-an-azure-sku)
    - **P** - [Power BI Premium](https://learn.microsoft.com/power-bi/enterprise/service-admin-premium-purchase)

### Power BI Desktop

1. [Verify minimum system requirements](https://docs.microsoft.com/power-bi/fundamentals/desktop-get-the-desktop#minimum-requirements) are met
2. Download and install Power BI Desktop from one of the options below:
    - [Microsoft Store](https://aka.ms/pbidesktop) *(Recommended - includes automatic updates)*
    - [Download Center](https://www.microsoft.com/en-us/download/details.aspx?id=58494) - Ensure you have the latest version installed
